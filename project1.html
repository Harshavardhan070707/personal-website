Secure Login System

Project Overview

This project implements a secure login system using AES and SHA-256 encryption algorithms. It ensures that user passwords are encrypted before storage and verifies user credentials during login. This system is designed to prevent unauthorized access and protect sensitive information.

Key Features

  • Encryption of user passwords with AES
  • Secure hash verification using SHA-256
  • Protection against brute force and dictionary attacks
